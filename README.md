# Cocktail Merge Challenge - Playable Ad Demo for Playbox Team

## Overview

This project is a playable ad demo created for Playbox Team to showcase my skills in developing engaging, interactive ads that combine game development, UX design, and AI integration.

## Features

- **Engaging Gameplay**: Merge-style puzzle mechanics in a 30-second format optimized for playable ads
- **Visual Polish**: Animated elements, particle effects, and satisfying feedback
- **AI Integration**: Smart hint system that suggests optimal moves
- **Mobile Optimized**: Full touch support and performance optimizations
- **Professional Quality**: Clean code architecture and cross-platform compatibility

## Technologies Used

- **PixiJS**: For high-performance 2D rendering
- **HTML5/CSS3/JavaScript**: For responsive web experience
- **AI Integration**: Algorithmic hint system for intelligent gameplay suggestions

## Files Included

- `index.html` - Main HTML structure with responsive UI
- `script.js` - Game logic, rendering, and AI features
- `portfolio.md` - Detailed portfolio presentation of the development process

## How to Run

1. Open `index.html` in a web browser
2. The game will start automatically
3. Tap on two identical cocktails to merge them
4. Use the "AI Hint" button to get intelligent move suggestions
5. Try to achieve the highest score in 30 seconds
